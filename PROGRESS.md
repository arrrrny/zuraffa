**Example:**
```dart
final parser = JsonParser();
final schema = parser.parseJson({
  'id': 'prod-123',
  'name': 'iPhone',
  'price': 999.99,
  'inStock': true,
  'tags': ['phone', 'electronics'],
  'createdAt': '2025-11-14T12:00:00Z',
});

print(schema.fields[0].type); // String
print(schema.fields[2].type); // double
print(schema.fields[5].type); // DateTime
```
