## 🏗️ Architecture Decisions Made

1. **Primitives Only Forever:** No smart type inference (Money, Email, etc.)
2. **Master Branch:** Using `master` instead of `main`
3. **$ References:** Following zikzak_morphy conventions exactly
4. **TDD First:** Tests before implementation, always
5. **Zero Dependencies:** Core package has minimal deps (args, meta, path)
