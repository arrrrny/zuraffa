## Submit Your App

Want to showcase your app built with Zuraffa? We'd love to feature it!

### Requirements

Your app must:
- Use Zuraffa as the primary architecture framework
- Implement at least 3 major Zuraffa patterns (entity-based, single repository, orchestrator, or polymorphic)
- Be publicly available (app store, web, or demo)
- Have a clear description of how Zuraffa enhanced your development

### How to Submit

1. **Prepare Information**
   - App name and description
   - Link to app (store page, website, or demo)
   - Download/install count (if available)
   - Brief description of Zuraffa features used
   - Screenshots or demo video (optional)

2. **Create a Pull Request**
   - Fork the Zuraffa repository
   - Add your app to the showcase table in `website/docs/built-on-zuraffa/showcase.md`
   - Submit a pull request with your changes

3. **Alternative: Open an Issue**
   - Visit our [GitHub Issues](https://github.com/arrrrny/zuraffa/issues)
   - Create a new issue with the title "App Showcase Submission: [Your App Name]"
   - Include all required information in the issue description

### Example Submission Format

```markdown
### [AppName]
**Downloads:** [Count]  
**Link:** [URL]  
**Description:** [Brief description of the app and how Zuraffa was used]

**Zuraffa Features Used:**
- [Feature 1]
- [Feature 2]
- [Feature 3]
```
